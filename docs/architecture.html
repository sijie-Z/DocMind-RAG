🎯 项目定位Positioning

DocMind 是一个企业级 AI Agent 平台，以 ReAct Agent 为核心，融合混合检索 RAG、知识图谱、工作流引擎。 支持多租户组织隔离、RBAC 权限体系、SSE 流式交互，覆盖文档管理、智能问答、深度分析、自动化工作流等场景。

DocMind is an enterprise AI Agent platform powered by a ReAct Agent core, integrating hybrid RAG, knowledge graphs, and a workflow engine. Supports multi-tenant isolation, RBAC, SSE streaming interaction, covering document management, Q&A, deep analysis, and automated workflows.

📐 设计原则Design Principles
  • → 模块化：解耦 Agent / RAG / 工作流，独立演进Modular: decoupled Agent / RAG / Workflow
  • → 异步优先：FastAPI async + SQLAlchemy 2.0 asyncAsync-first: FastAPI + SQLAlchemy 2.0 async
  • → 可观测：OpenTelemetry 全链路追踪 + Prometheus 指标Observable: OpenTelemetry tracing + Prometheus metrics
  • → 安全纵深：JWT + RBAC + 多租户 + 审计日志Defense in depth: JWT + RBAC + multi-tenancy + audit
