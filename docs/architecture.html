🎯 项目定位

DocMind 是一个企业级 AI Agent 平台，以自研 PER Agent（Plan-Execute-Reflect）为核心，融合混合检索 RAG、知识图谱、工作流引擎。支持多租户组织隔离、RBAC 权限体系、SSE 流式交互，覆盖文档管理、智能问答、深度分析、自动化工作流等场景。

📐 设计原则
  • → 模块化：解耦 Agent / RAG / 工作流，独立演进
  • → 异步优先：FastAPI async + SQLAlchemy 2.0 async
  • → 可观测：OpenTelemetry 全链路追踪 + Prometheus 指标
  • → 安全纵深：JWT + RBAC + 多租户 + 审计日志

▶ 📥 文档处理流程
上传→ MIME 校验→ SHA-256 去重→ MinIO 存储→ Kafka 通知→ Worker 消费
LangChain 解析→ 智能分块→ 向量化→ ES 索引
▶ 🔎 检索策略
策略召回方式权重
BM25关键词精确匹配0.4
KNN 向量语义相似度0.6
RRF 融合倒数排序融合—
Rerank智谱 rerank 重排序—

默认取 Top-10 检索结果，Rerank 后取 Top-5 送入 LLM

基于向量相似度的查询缓存，相同语义问题秒级响应（< 50ms），缓存命中率约 30%。
