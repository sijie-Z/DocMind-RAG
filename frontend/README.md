# 派聪明AI知识库系统

基于RAG架构的企业级AI知识库智能问答平台

## 技术栈

### 后端
- **框架**: FastAPI + Python 3.8+
- **数据库**: MySQL 8.0 + SQLAlchemy
- **缓存**: Redis
- **搜索引擎**: Elasticsearch 8.10.0
- **消息队列**: Apache Kafka
- **文件存储**: MinIO
- **AI集成**: DeepSeek API + 豆包Embedding
- **安全认证**: JWT + 密码加密(BCrypt)

### 前端
- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **UI组件**: Naive UI
- **状态管理**: Pinia
- **路由**: Vue Router
- **样式**: UnoCSS + SCSS
- **图标**: Iconify

## 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- MySQL 8.0
- Redis 6.0+
- Elasticsearch 8.10.0
- Apache Kafka
- MinIO

### 后端启动
```bash
# 安装依赖
pip install -r requirements.txt

# 启动后端服务
cd paicongming_backend
python main.py

# 启动文件处理服务
python file_processor.py
```

### 前端启动
```bash
# 安装依赖
cd paicongming_frontend
npm install

# 启动开发服务器
npm run dev
```

## API文档

### 认证接口
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `GET /api/user/profile` - 获取用户信息

### 文件接口
- `POST /api/files/upload/init` - 初始化文件上传
- `POST /api/files/upload/chunk` - 上传文件分片
- `GET /api/files` - 获取文件列表

### 聊天接口
- `POST /api/chat` - 发送消息
- `WS /ws/chat/{user_id}` - WebSocket聊天

### 组织接口
- `POST /api/organizations` - 创建组织
- `GET /api/organizations` - 获取组织列表

## 核心算法

### RAG检索算法
1. **文档预处理**: 支持20+种文件格式解析
2. **文本分块**: 智能文本分割，块大小512字符
3. **向量生成**: 768维向量嵌入
4. **混合检索**: 语义检索 + 关键词检索 + 重排序
5. **上下文构建**: 动态Prompt构建

### 文件处理流程
1. **分片上传**: 5MB分片，支持并发上传
2. **异步处理**: Kafka消息队列处理
3. **文档解析**: Apache Tika集成
4. **向量化**: 豆包Embedding API
5. **索引构建**: Elasticsearch存储
